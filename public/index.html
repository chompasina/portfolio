Denver Recycling Request Tool

I worked with a team of volunteer developers at Code for Denver to create this open source request platform for tenants of multi-family buildings within the City of Denver to transparently and collectively petition their landlord to add recycling collection services. This app uses a Rails API with a PostgresQL database, React frontend with Redux state management, Google Places API, and is styled with Sass. Once the app is complete, we’ll create a customizable version that citizens in any city in the world can use to request services from their landlord.

  • Client: Code for Denver & EcoCycle
  • Date: 2014-07-05
  • Category: Web Development

Lending Hands Microlending Services Platform

This brownfield project pivoted a simple e-commerce store into a multitenancy microlending platform. The app was developed with MVC architecture using Rails, database transactions and logic operations using ActiveRecord, user authentication and authorization, a functional shopping cart with Stripe integration, different levels of authorization (registered user, organization admin, platform admin), and Javascript/jQuery methods for search/filter and cart updating features.

  • Date: 2014-07-07
  • Category: Web Development

Rails Engine

This project uses Rails and ActiveRecord to build a versioned JSON API which exposes the SalesEngine data schema. The RailsEngine data was imported from the SalesEngine CSV files by building rake tasks. We used controller tests to drive the design, and Ruby and ActiveRecord to perform more complicated business intelligence queries.

  • Date: 2014-07-08
  • Category: Web Development

Anonymous Job SlackBot

I built this Slackbot to help people anonymously share their current job status for the purposes of data collection. The app uses the Slack API with configured slash commands to authenticate users against a PostgresQL database of confirmed authorized users, and stores their data privately in a Rails backend, without the user leaving Slack.

  • Date: 2014-07-10
  • Category: Web Development
